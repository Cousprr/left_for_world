# linux 命令行常用命令

列出当前文件目录

```
ls
```

转到指定目录

```
cd ~/dir
```

退回到根目录

```
cd
```

创建目录

```
mkdir dir_name
```

删除目录

```
rm -r dir
```

删除文件

```
rm file_name
```

复制文件

```
cp file_name target_dir
```

移动文件

```
mv file_name dir
```

查看文件内容
```
cat file_name
```
