## 写在前面的话

本仓库是对本人在ucas挣扎求生经历期间写的一些小而细碎的知识点和踩过的坑的一份归档整理，随心而感，不成体系。如果你无意中点进来，可以逛一逛感兴趣的部分，也许你能从里面得到一点微不足道的帮助。

需要说明的是本人水平低于本校平均水平，因此如果你抱着求高分的心态，很抱歉，我更推荐你去参考更厉害的同学。本文档只面向相同水平的受众，或哪些想大略了解计算机系本科研讨课的朋友。

如有谬误之处，欢迎斧正，不胜感激。当然如果只是想单纯和作者聊聊，也欢迎提issue或私信我的邮件`cousprr3517@163.com`。
